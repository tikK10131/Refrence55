# サンプルデータ

## 1. 目的
- 開発・検証環境で VRView の主要機能（施設ツリー、VR画像、地図、付箋、設備、権限）を即時確認できる初期データを投入する。
- サンプルデータ仕様の正本は本MDとし、SQLは本MDに準拠して作成する。

## 2. 適用手順
1. `DB設計/schema.sql` を適用する。
2. `DB設計/sample_data.sql` を実行する。
3. 主要シーケンスは `sample_data.sql` 末尾の `setval` で再調整される。

## 5. 備考
- 本データは論理削除済みレコードを含まない。
- 画像・PDF本体はMinIOに配置し、ブラウザからは常にMock API (`/api/files/:fileId`) 経由で取得する。
- Mock API が MinIO から取得して代理配信するため、ブラウザは `:8000` のみ疎通できればよい（`:9000` の直接公開は不要）。
- サンプル値（ファイルURL、作成日時、コメント文言など）は検証目的で固定している。
